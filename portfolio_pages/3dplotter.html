Web based 3D Differental Equations Plotter

Plotting systems of 3 equations using Eulers method on the web

As I was scrolling around the internet one day I came across a very cool (and nerdy) website that contained web based demonstrations of many Strange Attractors from a website called DynamicMath.xyz . Below you will find a small snipbit of how they displayed them

After seeing this I was quite inspired and decided to make my own attempt at making something like this. Sense it combined two of my great intrests, and beacause I need to get some experience coding in the web, I decided i would make my own web based version from the ground up.

The system works by using Eulers Method To calcuate where the solution of the system of equations (based on given starting conditions) would go without explicitly finding the solution to the system. Then it calcuates where the solution might go for a certain number of iterations stores that data and uses it to make the display.
